## рџ§  Scoping вЂ” LEGB Rule

Variables are resolved using the **LEGB** (Local в†’ Enclosing в†’ Global в†’ Built-in) scoping chain.

Each function call creates a new **LexicalEnvironment** with a pointer to its parent.
This enables closures вЂ” inner functions can access outer function variables even after the outer function returns.
